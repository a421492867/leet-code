### 动态规划

> 动态规划在查找有很多重叠子问题的情况的最优解时最有效
> 只能应用于有最优子结构的问题  最优子结构指的是 局部最优解能决定全局最优解
> 保存子问题的解 避免重复计算
> 关键是找到状态转移方程
> 一些情况下可看作带有状态记录的优先搜索
> 
> 
> 一维  二维  分割问题  子序列问题   背包问题    字符串编辑  股票交易
